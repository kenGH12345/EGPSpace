# 摩擦力实验审核

## 物理概念
- 摩擦力：两个相互接触的物体发生相对运动或有相对运动趋势时产生的阻碍力。
- 静摩擦力：物体静止时的摩擦力，大小随外力变化（≤ f_max）。
- 滑动摩擦力：物体滑动时的摩擦力，大小恒定 f_k = μ_k·N。
- 最大静摩擦力：f_max = μ_s·N，通常略大于滑动摩擦力。

## 实验目的
- 探究正压力、接触面粗糙程度对摩擦力的影响。
- 理解静摩擦→滑动摩擦的临界突变现象。
- 通过 f-F 图像直观展示摩擦力随拉力的变化规律。

## 操作步骤
1. 选择接触面类型：木板、玻璃或毛巾（预设摩擦系数）。
2. 调节正压力 N 和拉力 F。
3. 观察 Canvas 中木块状态和弹簧测力计示数。
4. 逐步增大拉力，观察从"静止"→"即将滑动"→"滑动中"的状态变化。
5. 查看右侧 f-F 关系图，理解平台→突降的物理意义。

## 配套公式
```
f_max = μ_s · N       (最大静摩擦力)
f_k = μ_k · N         (滑动摩擦力)
静摩擦时：f = F_拉    (静摩擦力等于拉力)
```

## 变量与结果
| 变量 | 范围 | 说明 |
|------|------|------|
| 正压力 N | 10 ~ 200 N | 垂直于接触面的压力 |
| 拉力 F | 0 ~ 150 N | 水平拉力 |
| 静摩擦系数 μ_s | 0.05 ~ 1.00 | 静摩擦特性 |
| 滑动摩擦系数 μ_k | 0.03 ~ 0.90 | 滑动摩擦特性 |

**结果记录**: 当前摩擦力(N)、最大静摩擦(N)、滑动摩擦(N)、运动状态。
