# 常用物理公式汇总

## 一、力学

### 1. 运动学

| 公式 | 适用条件 |
|------|---------|
| v = s/t | 匀速直线运动 |
| a = (v₂ - v₁)/t | 加速度定义 |
| v = v₀ + at | 匀变速直线运动 |
| s = v₀t + ½at² | 匀变速直线运动位移 |
| v² - v₀² = 2as | 匀变速直线运动（不含时间） |

### 2. 动力学

| 公式 | 说明 |
|------|------|
| F = ma | 牛顿第二定律 |
| G = mg | 重力计算 |
| f = μN | 滑动摩擦力 |
| F₁ × L₁ = F₂ × L₂ | 杠杆平衡条件 |

### 3. 浮力

| 公式 | 说明 |
|------|------|
| F浮 = ρ液 × g × V排 | 阿基米德原理（定义式）|
| F浮 = G - F拉 | 称重法测浮力 |
| F浮 = G排 | 浮力等于排开液体重量 |

### 4. 浮沉条件

| 条件 | 状态 | 关系 |
|------|------|------|
| ρ物 < ρ液 | 上浮（最终漂浮） | F浮 > G |
| ρ物 = ρ液 | 悬浮 | F浮 = G |
| ρ物 > ρ液 | 下沉 | F浮 < G |

## 二、光学

### 1. 折射定律

```
n₁ × sin(θ₁) = n₂ × sin(θ₂)
```

- n₁：入射介质折射率
- θ₁：入射角（法线与光线夹角）
- n₂：折射介质折射率
- θ₂：折射角

### 2. 临界角

```
C = arcsin(1/n)  （n > 1，从光密到光疏）
```

| 介质 | 折射率 | 对空气临界角 |
|------|--------|-------------|
| 水 | 1.33 | 48.75° |
| 玻璃 | 1.50 | 41.8° |
| 钻石 | 2.42 | 24.4° |

### 3. 透镜成像

| 公式 | 说明 |
|------|------|
| 1/f = 1/u + 1/v | 透镜成像公式 |
| m = |v|/u | 放大率 |

## 三、电学

### 1. 基本定律

| 公式 | 说明 |
|------|------|
| I = Q/t | 电流定义 |
| I = U/R | 欧姆定律 |
| R = ρL/S | 电阻定律 |
| P = UI | 电功率 |
| P = I²R | 纯电阻电功率 |

### 2. 串联电路

| 物理量 | 关系 |
|--------|------|
| 电流 | I = I₁ = I₂ = ... |
| 电压 | U = U₁ + U₂ + ... |
| 电阻 | R = R₁ + R₂ + ... |
| 分压 | U₁/U₂ = R₁/R₂ |

### 3. 并联电路

| 物理量 | 关系 |
|--------|------|
| 电流 | I = I₁ + I₂ + ... |
| 电压 | U = U₁ = U₂ = ... |
| 电阻 | 1/R = 1/R₁ + 1/R₂ + ... |
| 分流 | I₁/I₂ = R₂/R₁ |

## 四、热学

### 1. 热量计算

| 公式 | 说明 |
|------|------|
| Q = cm(t₂ - t₁) | 物体吸放热 |
| Q = mq | 燃料燃烧放热 |
| Q = mλ | 熔化（或凝固）放热 |
| Q = mL | 汽化（或液化）放热 |

### 2. 热效率

```
η = Q有用/Q总 × 100%
```

## 五、化学（酸碱滴定）

### 1. pH 定义

```
pH = -lg[H⁺]
```

### 2. 水的离子积

```
Kw = [H⁺][OH⁻] = 1×10⁻¹⁴  (25°C)
pH + pOH = 14
```

### 3. 中和反应

```
n(H⁺) = n(OH⁻)  （等当点）
```

### 4. 浓度计算

| 公式 | 说明 |
|------|------|
| c = n/V | 物质的量浓度 |
| n = m/M | 物质的量与质量 |
| c₁V₁ = c₂V₂ | 稀释公式（等当点）|

## 八、公式使用注意事项

### 力学
- 使用 F = ma 时，m 必须是质量（kg），F 必须是合外力（N）
- 浮力公式中的 V排 是物体浸入液体的体积，不是物体体积
- 杠杆平衡时，力臂是支点到力的作用线的垂直距离

### 光学
- 入射角和折射角都是与法线的夹角，不是与界面的夹角
- 斯涅尔定律中，角度要用弧度或度数均可，但两边单位要统一
- 全反射只发生在光从光密介质射向光疏介质时

### 电学
- 欧姆定律适用于纯电阻电路
- 串并联时，注意分析电路结构
- 功率公式在不同条件下选择使用

### 热学
- 注意比热容的单位
- 物态变化时的热量计算要区分过程

### 化学
- pH 计算中，[H⁺] 单位是 mol/L
- 等当点附近 pH 变化剧烈，中和滴定曲线在等当点有突跃
