# 生物学科知识库

## 细胞结构

### 动物细胞 vs 植物细胞

| 结构 | 动物细胞 | 植物细胞 |
|------|---------|---------|
| 细胞壁 | 无 | 有（纤维素）|
| 叶绿体 | 无 | 有 |
| 液泡 | 小（不明显）| 大（中央液泡）|
| 中心体 | 有 | 无（低等植物有）|

### 细胞器功能

| 细胞器 | 功能 |
|--------|------|
| 线粒体 | 有氧呼吸的主要场所，"能量工厂" |
| 叶绿体 | 光合作用的场所，"养料制造车间" |
| 内质网 | 蛋白质合成加工运输 |
| 高尔基体 | 蛋白质加工分泌 |
| 核糖体 | 蛋白质合成 |
| 溶酶体 | 分解衰老细胞器 |
| 液泡 | 调节渗透压，储存物质 |

## 新陈代谢

### 光合作用

**公式**：
```
CO₂ + H₂O → (CH₂O) + O₂
      光能、叶绿体
```

**阶段**：
| 阶段 | 场所 | 物质变化 | 能量变化 |
|------|------|---------|---------|
| 光反应 | 类囊体薄膜 | H₂O → O₂ | 光能 → ATP |
| 暗反应 | 叶绿体基质 | CO₂ → C₃ → C₆H₁₂O₆ | ATP → 有机物 |

**影响因素**：
- 光照强度（光饱和点）
- CO₂浓度
- 温度
- 水分
- 必需矿质元素

### 细胞呼吸

**有氧呼吸**：
```
C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 能量
```

| 阶段 | 场所 | 产物 |
|------|------|------|
| 第一阶段 | 细胞质基质 | 丙酮酸 + [H] + 2ATP |
| 第二阶段 | 线粒体基质 | CO₂ + [H] + 2ATP |
| 第三阶段 | 线粒体内膜 | H₂O + 34ATP |

**无氧呼吸**：
```
植物/酵母菌：C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂ + 能量
动物/乳酸菌：C₆H₁₂O₆ → 2C₃H₆O₃ + 能量
```

### 光合作用 vs 细胞呼吸

| 项目 | 光合作用 | 细胞呼吸 |
|------|---------|---------|
| 场所 | 叶绿体 | 线粒体、细胞质 |
| 条件 | 光 | 有光无光均可 |
| 原料 | CO₂ + H₂O | O₂ + 有机物 |
| 产物 | 有机物 + O₂ | CO₂ + H₂O |
| 能量 | 储存能量 | 释放能量 |
| 实质 | 合成代谢 | 分解代谢 |

## 遗传与变异

### 孟德尔遗传定律

**分离定律**：
- 成对基因在配子形成时彼此分离
- F₂比例：3:1（显性:隐性）

**自由组合定律**：
- 非等位基因独立分配
- F₂基因型9:3:3:1

### DNA 结构

| 项目 | 内容 |
|------|------|
| 基本单位 | 脱氧核苷酸 |
| 双链结构 | 反向平行 |
| 碱基配对 | A=T, G≡C |
| 主要元素 | C, H, O, N, P |

### 中心法则

```
DNA → RNA → 蛋白质
  ↘      ↗
   逆转录（某些病毒）
```

### 变异类型

| 类型 | 原因 | 特点 |
|------|------|------|
| 基因突变 | DNA分子改变 | 根本来源 |
| 基因重组 | 基因重新组合 | 可遗传变异 |
| 染色体变异 | 数目/结构变化 | 可肉眼观察 |

**植物吸水方式**：

- 渗透作用（成熟细胞）
- 主动运输（矿质离子）

## 常见错误分析

### 概念错误
| 错误 | 正确理解 |
|------|---------|
| "光合作用需要光，呼吸作用不需要光" | 呼吸作用有光无光均可进行 |
| "基因突变都是有害的" | 有些突变是有利的或中性的 |
| "变异都能遗传" | 只有生殖细胞的变异才能遗传 |
| "植物不需要氧气" | 植物呼吸作用需要O₂ |

### 实验错误
| 错误 | 正确做法 |
|------|---------|
| 观察植物细胞用清水 | 用蒸馏水（保持渗透压）|
| 洋葱表皮直接观察 | 先制作装片 |
| 脂肪检测直接加热 | 苏丹III染色后需50%酒精洗浮色 |

### 计算错误
| 错误 | 正确理解 |
|------|---------|
| DNA复制n次后碱基数 | 2^n 个DNA分子 |
| PCR扩增 | n次循环后为2^n倍 |
