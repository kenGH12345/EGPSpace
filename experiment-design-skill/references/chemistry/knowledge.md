## 反应类型

### 四种基本反应类型

| 类型 | 表达式 | 示例 |
|------|--------|------|
| 化合反应 | A + B → AB | 2H₂ + O₂ → 2H₂O |
| 分解反应 | AB → A + B | 2H₂O → 2H₂ + O₂ |
| 置换反应 | A + BC → AC + B | Zn + 2HCl → ZnCl₂ + H₂↑ |
| 复分解反应 | AB + CD → AD + CB | NaOH + HCl → NaCl + H₂O |

### 氧化还原反应

**特征**：化合价有升降（电子转移）

| 概念 | 定义 |
|------|------|
| 氧化反应 | 失去电子（化合价升高）|
| 还原反应 | 得到电子（化合价降低）|
| 氧化剂 | 得到电子的物质（被还原）|
| 还原剂 | 失去电子的物质（被氧化）|

**常见氧化剂**：O₂, Cl₂, KMnO₄, H₂SO₄(浓), HNO₃
**常见还原剂**：C, CO, H₂, 金属单质

### 离子反应

**条件**：生成沉淀、气体、水（弱电解质）

**常见沉淀**：
- 白色：CaCO₃, BaCO₃, AgCl, BaSO₄, Mg(OH)₂
- 蓝色：Cu(OH)₂
- 红褐色：Fe(OH)₃
- 黑色：CuO, Fe₃O₄, Ag₂O

## pH 与指示剂

### pH 范围
- pH < 7：酸性
- pH = 7：中性
- pH > 7：碱性

### 常见指示剂
| 指示剂 | 酸性 | 中性 | 碱性 |
|--------|------|------|------|
| 石蕊 | 红色 | 紫色 | 蓝色 |
| 酚酞 | 无色 | 无色 | 红色 |
| 甲基橙 | 红色 | 橙色 | 黄色 |
| 甲基红 | 红色 | 橙色 | 黄色 |

### pH 计算
```
pH = -lg[H⁺]
[H⁺] = 10^-pH
pH + pOH = 14 (25°C)
Kw = [H⁺][OH⁻] = 10⁻¹⁴
```

## 化学计量

### 基本公式
| 公式 | 说明 |
|------|------|
| n = m/M | 物质的量与质量 |
| n = V/22.4 | 标况气体体积 |
| n = CV | 溶液物质的量 |
| m = ρV | 质量计算 |
| w = m质/m液 × 100% | 质量分数 |

### 溶液配制
```
C₁V₁ = C₂V₂  (稀释公式)
m₁w₁ = m₂w₂  (浓缩公式)
```

## 常见错误分析

### 概念错误
| 错误 | 正确理解 |
|------|---------|
| "催化剂增加产物量" | 催化剂不改变化学平衡，只改变反应速率 |
| "燃烧都需要氧气" | 燃烧是剧烈氧化还原反应，不一定有O₂ |
| "酸都能腐蚀金属" | 只有活泼金属才能与酸反应 |
| "盐都是中性" | 盐溶液可能酸性或碱性（如Na₂CO₃碱性）|

### 操作错误
| 错误 | 正确做法 |
|------|---------|
| 稀释浓硫酸时向水中加酸 | 应将酸沿器壁慢慢加入水中，并不断搅拌 |
| 倾倒液体时标签朝下 | 标签应朝手心，防止液体流下腐蚀标签 |
| 用嘴吹灭酒精灯 | 应用灯帽盖灭 |
| 加热液体超过容积1/3 | 液体不超过试管容积的1/3 |

### 计算错误
| 错误 | 正确理解 |
|------|---------|
| 漏写气体/沉淀符号 | ↑↓符号要标注 |
| 配平后原子数不等 | 质量守恒检验 |
| 浓度单位混淆 | mol/L ≠ g/mL |

## 实验安全

### 危险化学品标志
- 爆炸品
- 易燃品
- 氧化剂
- 剧毒品
- 腐蚀品

### 紧急处理
| 情况 | 处理方法 |
|------|---------|
| 酸洒在皮肤上 | 立即用大量水冲洗，再用NaHCO₃溶液冲洗 |
| 碱洒在皮肤上 | 立即用大量水冲洗，再用硼酸冲洗 |
| 酒精灯打翻 | 用湿抹布覆盖 |
| 着火 | 切断电源，用沙土/灭火器扑灭 |
