- **内容一致性**：
  - ✅ Q = cmΔT 公式正确

## 3. 交互设计

### 3.1 用户旅程
1. 默认状态：t=0s，两柱均显示 T=20°C
2. 拖动时间滑块 → 两柱高度差逐渐拉开（铝升温更快）
3. 交换比热容数值 → 升温快慢互换
4. 调整质量/功率观察对温升的影响

### 3.2 交互元素
- **比热容双滑块**：A/B 各自可调 0.2~2.5 J/g°C
- **质量滑块**：50~500g
- **功率滑块**：50~1000W
- **时间滑块**：0~300s
- **Canvas**：双柱温度可视化 + mini 温度-时间曲线

### 3.3 边界情况
- t = 0：ΔT = 0，两柱等高
- c → 0：理论上 ΔT → ∞，实际 slider 下限 0.2
- P = 50, m = 500, c = 2.5, t = 300 → ΔT = 12°C（合理范围）
