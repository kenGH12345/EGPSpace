## 1. 学科一致性

### 1.1 教材标准
- **人教版**：八年级上册第二章「声现象」第一节「声音的产生与传播」、第二节「声音的特性」
- **内容一致性**：
  - ✅ 音调由频率决定：频率越高，音调越高
  - ✅ 响度由振幅决定：振幅越大，响度越大
  - ✅ 波速公式：v = fλ（空气中 v ≈ 343 m/s）
  - ✅ 人耳听觉范围：20Hz~20000Hz（slider 上限 2000Hz 为初中简化范围）

### 1.2 实验现象
- 拖动频率滑块 → 波形压缩/拉伸，音调标签从「低音」到「高音」变化
- 拖动振幅滑块 → 波形高度变化，粒子大小变化
- 粒子动画展示声波在空气中的传播
- 静态波形快照（金色线）叠加在动画上方

### 1.3 常见问题对照
- 学生常见误区：音调和响度混淆 → 界面分别标注「频率」和「振幅」并给出音调文字提示
- 学生常见误区：认为波长与频率无关 → 实时计算并显示 λ = v/f

## 3. 交互设计

### 3.1 用户旅程
1. 默认 f=440Hz, A=1.0 → 看到标准波形，类似钢琴 A4 音
2. 增大频率 → 波形压缩，音调标签变为「高音区」
3. 减小频率 → 波形拉伸，音调标签变为「低音区」
4. 调整振幅 → 波形高度和粒子大小同步变化

### 3.2 交互元素
- **频率滑块**：20~2000Hz
- **振幅滑块**：0.1~2.0
- **Canvas 动画**：粒子波动 + 波形快照叠加
- **音调标签**：实时文字描述（低音/中音/高音区）

### 3.3 边界情况
- f = 20Hz：接近人耳下限，波长最长（约 17m）
- f = 2000Hz：高频，波长最短（约 0.17m）
- A = 0.1：极小振幅，几乎静音
- A = 2.0：大振幅，响度大
