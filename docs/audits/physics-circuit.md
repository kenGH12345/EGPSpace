## 1. 物理原理

### 串联电路
- **总电阻**：`R串 = R₁ + R₂`
- **总电流**：`I = U / R串`（各电阻电流相同）
- **分压**：`U₁ = I × R₁`，`U₂ = I × R₂`，`U₁ + U₂ = U`

### 并联电路
- **总电阻**：`1/R并 = 1/R₁ + 1/R₂` 即 `R并 = R₁R₂ / (R₁+R₂)`
- **各支路电压**：`U₁ = U₂ = U`（电源电压）
- **各支路电流**：`I₁ = U/R₁`，`I₂ = U/R₂`，`I总 = I₁ + I₂`

### 功率
- `P = U × I`
