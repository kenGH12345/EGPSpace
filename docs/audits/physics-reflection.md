## 1. 学科一致性

### 1.1 教材标准
- **人教版**：八年级上册第四章「光现象」第二节「光的反射」
- **内容一致性**：
  - ✅ 反射定律：入射角 = 反射角（θᵢ = θᵣ）
  - ✅ 法线垂直于反射面
  - ✅ 入射光线、反射光线、法线在同一平面内
  - ❌ 未 3D 展示三线共面（仅展示了 2D 截面）

### 1.2 实验现象
- 当入射角从 0° 增加到 89° 时，反射角严格等于入射角
- 法线与镜面夹角始终为 90°
- 不涉及漫反射与镜面反射的区分

### 1.3 常见问题对照
- 学生常见误区：认为入射光线与镜面的夹角是入射角 → 已在界面上方标注法线和角度弧线

## 3. 交互设计

### 3.1 用户旅程
1. 默认入射角 45°，看到对称的入射/反射光线
2. 拖动滑块改变入射角 → 光线实时跟随，角度弧线动态重绘
3. 观察入射角 = 反射角的恒等关系

### 3.2 交互元素
- **入射角滑块**：0°~89°，步进 1°，实时更新
- **光线图**：SVG 绘制，包含入射光线（金色）、反射光线（蓝色）、法线（虚线灰色）
- **角度标注**：弧线 + 统计面板

### 3.3 边界情况
- θᵢ = 0°：光线垂直入射，沿法线返回
- θᵢ = 89°：接近掠入射，光线几乎贴镜面
