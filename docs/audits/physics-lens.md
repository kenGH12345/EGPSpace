## 1. 学科一致性

### 1.1 教材标准
- **人教版**：八年级上册第五章「透镜及其应用」第二节「生活中的透镜」、第三节「凸透镜成像的规律」
- **内容一致性**：
  - ✅ 高斯透镜公式：1/f = 1/u + 1/v
  - ✅ 物距 u > 2f：倒立、缩小、实像
  - ✅ f < u < 2f：倒立、放大、实像
  - ✅ u < f：正立、放大、虚像
  - ✅ 三条典型光线追踪（平行轴→过F'、过光心直行、过F→平行轴）

### 1.2 实验现象
- 拖动焦距滑块 → 透镜曲率变化，F/2F 标记自动移动
- 拖动物距滑块 → 像距、放大率、像的性质实时更新
- 当 u = f 时，提示「像在无穷远」（光线平行射出）

### 1.3 常见问题对照
- 学生常见误区：虚像也在透镜另一侧 → 界面上明确标注「虚像」并显示在物体同侧
- 学生常见误区：u = f 时有像 → 公式给出 v = ∞，已在界面提示

## 3. 交互设计

### 3.1 用户旅程
1. 默认 f=15cm, u=40cm → 看到倒立缩小实像
2. 减小物距到 u=20cm (u=2f) → 像距 v=20cm，等大倒立实像
3. 继续减小到 u=10cm (u<f) → 突变：变为正立放大虚像
4. 调整焦距观察不同透镜的成像差异

### 3.2 交互元素
- **焦距滑块**：5~40cm，步进 1cm
- **物距滑块**：10~100cm，步进 1cm
- **SVG 光线图**：三条典型光线 + 物体/像箭头 + F/2F 标记
- **性质标签**：实时显示「实像/虚像」「放大/缩小」「正立/倒立」

### 3.3 边界情况
- u = f：v = ∞，界面提示「像在无穷远」
- u → 0：v → 0，像趋近于透镜
- f = 0：物理上无意义，slider 下限为 5cm
